# 通过 flex 布局，以三栏作为 flex item 子元素

center设置 flex 为 1 ，两边设置绝对px

flex 为 1 等于 110

1 flex-grow 有剩余空间时，按比例分配

1 flex-shrink 空间不足时，允许压缩

0 flex-basis 初始（一般是auto跟随width

```tsx
<style>
.container {
  display: flex;
}
.left {
  width: 200px;
  background: lightblue;
}
.center {
  flex: 1; /* 自适应宽度 */
  background: lightgreen;
}
.right {
  width: 200px;
  background: lightcoral;
}
</style>

<div class="container">
  <div class="left">左</div>
  <div class="center">中</div>
  <div class="right">右</div>
</div>
```

# 基于Grid

**grid-template-columns 设置列宽，（**

**grid-template-rows 设置行高）**

```tsx
<style>
.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
}
.left {
  background: lightblue;
}
.center {
  background: lightgreen;
}
.right {
  background: lightcoral;
}
</style>

<div class="container">
  <div class="left">左</div>
  <div class="center">中</div>
  <div class="right">右</div>
</div>
```

# 顶栏吸附

写一个与内容div平齐的header然后 position 设置 sticky

```tsx
<style>
body, html {
  margin: 0;
  height: 100%;
}

/* 吸顶导航栏 */
header {
  position: sticky;
  top: 0;
  background: #333;
  color: white;
  padding: 15px;
  font-size: 18px;
  z-index: 100;
}

/* 三栏 Grid */
.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  height: 100%;
}

/* 示例样式 */
.left {
  background: lightblue;
  padding: 20px;
}
.center {
  background: lightgreen;
  padding: 20px;
  height: 2000px; /* 模拟滚动内容 */
}
.right {
  background: lightcoral;
  padding: 20px;
}
</style>

<body>
  <header>吸顶导航栏</header>

  <div class="container">
    <div class="left">左侧栏</div>
    <div class="center">中间内容（长内容以测试滚动）</div>
    <div class="right">右侧栏</div>
  </div>
</body>
```
